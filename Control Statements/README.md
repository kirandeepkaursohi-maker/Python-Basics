# 🔀 Python Control Statements

This folder contains programs demonstrating different types of **control statements** used to make decisions in Python.

---

## 📚 Concepts Covered

* **if statement**
  Executes a block of code when a condition is true

* **if-else statement**
  Chooses between two conditions

* **if-elif-else statement**
  Handles multiple conditions

* **Nested if**
  An if statement inside another if

* **Ternary Operator**
  Short form of if-else in a single line

---

## 🎯 Purpose

* Learn how programs make decisions
* Build logical thinking
* Apply conditions to real-world problems
